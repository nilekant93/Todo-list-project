What is this?

"It's Your Day" is a to-do list app that allows you to create tasks for the day. While using this app, you have no worries about forgetting those important meetings or buying flowers for your favorite person. If you want to make every day as perfect as it can be, you should start using this app!

How it works?

You can do simply these things:

Button Feature Result
+ Button
  • Set title (required)
  • Set time
  • Set description
 Click add-button and let the magic happen!
Trashcan button Delete your tasks by clicking them Tasks will be removed and re-ordered by the "title number"
About-button You can open this nice looking pop-up page I think you got it...

Good to know also

  • I don't let you add a new task If you don't have at least a title ;)
  • When you create a new task you will receive also an ordernumber. If you have done for example task 5 and want to remove it, the order number will change on next tasks. (6 will became 5 and so on...)
  • YOU DON'T LIKE THE BACKROUND IMAGE?? Alright, come back later and the image will be changed based on time of the day! ;)
